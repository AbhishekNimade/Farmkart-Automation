Smart Verification Logic

The system doesn't just click buttons; it thinks. Before confirming any booking, the automation verifies:

  • Address Match: Checks customer city and state against pin codes.
  • Weight Check: compares volumetric weight against box dimensions.
  • AWB Validation: Ensures generated AWB numbers are valid before saving.

This "double-check" architecture ensures 99.9% booking accuracy.

Common Questions

How do I start the booking process?

Click on the "Order Booking" card, enter the row number where pending orders start in your Google Sheet, and click "Start Automation".

What if an order fails?

The system will log the specific error in the terminal view and on the Google Sheet status column. You can manually inspect or skip it.

Does it work in the background?

Yes, the automation runs on the server. However, keep the terminal view open to monitor progress logs.

Best Practices

Optimization Tips

  • Check Connectivity: Ensure a stable internet connection for long-running scripts.
  • Verify Sheet Names: Don't rename Google Sheet tabs while the process is active.
  • Monitor Logs: Keep the terminal view open to catch any specific address validation failures.
  • Start Small: For first-time users, try running 5-10 rows to verify the configuration.
